# Authentication

## Get an API key

You can get an API key from [Settings > Integrations](https://typefully.com/?settings=integrations).

API keys let you use the API for a given ùïè / Twitter account you have connected to Typefully. Switch account from the account menu to generate keys for another account.

## Authorizing API requests

Include this header in all the requests you make:

| Header | Value |
|--------|-------|
| X-API-KEY | Bearer `TYPEFULLY_API_KEY` |
